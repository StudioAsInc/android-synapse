# Product Overview

Synapse is an open-source social media platform for Android that prioritizes privacy, real-time communication, and a lightweight user experience. Built with Kotlin and powered by Supabase backend.

## Core Features

- **Real-time Chat**: Direct messaging with secure Row Level Security (RLS) policies
- **Social Feed**: Create, view, like, and comment on posts with full user attribution
- **User Profiles**: View profiles, follow/unfollow users, manage social graph
- **Authentication**: Secure email and OAuth login via Supabase GoTrue
- **Media Storage**: 35GB+ free storage via Supabase Storage
- **Zero Ads**: Non-profit model prioritizing user experience

## Project Status

- Authentication, Chat, Posts, Feed, and Profiles are stable and production-ready
- Real-time updates and enhanced media handling are in development
- Group chats, video calls, and push notifications are planned

## Key Principles

- Privacy-first with end-to-end encryption for private chats
- Ad-free, non-profit model
- Lightweight and optimized for any device
- Open-source and community-driven
- MVVM architecture with Repository pattern
